# ☀️🌙 Day & Night Planner

나만의 하루를 완벽하게 기획하고 되돌아볼 수 있는 **미니멀 데일리 플래너**입니다. 
웹 브라우저 기반으로 동작하며, 복잡한 회원가입이나 서버 연동 없이 모든 데이터가 사용자의 기기에 안전하게 보관됩니다.

## ✨ 주요 기능
* **Day 모드 (아침/낮):** 오늘 하루의 일정(Schedule), 날씨, 핵심 할 일(Top Tasks), 챙길 물건, 스스로에게 전하는 다짐(Reminder)을 작성합니다.
* **Night 모드 (저녁/밤):** 오늘 먹은 식단(Today I ate), 기분(Mood Tracker), 일기(Journal), 그리고 내일을 위한 준비(Preparing tomorrow)를 기록합니다.
* **프라이빗 저장소:** 작성한 모든 내용은 브라우저의 로컬 스토리지(`localStorage`)에 저장되어 타인에게 절대 노출되지 않습니다.
* **캘린더 연동:** 달력의 날짜를 클릭하면 과거에 기록했던 Day & Night 데이터를 모달 창으로 다시 모아볼 수 있습니다.

## 🚀 설치 및 실행 방법 (Getting Started)

이 프로젝트는 순수 HTML, CSS, JavaScript로 작성되어 복잡한 환경 설정이 필요 없습니다. `VS Code`와 `Live Server` 확장 프로그램만 있으면 즉시 실행 가능합니다.

### 📋 사전 준비 (공통)
1. [Git](https://git-scm.com/) 설치
2. [Visual Studio Code (VS Code)](https://code.visualstudio.com/) 설치
3. VS Code의 Extensions(확장 프로그램) 탭에서 **`Live Server`** 검색 후 설치

### 🪟 Windows 환경
1. 명령 프롬프트(cmd) 또는 PowerShell을 열고 프로젝트를 다운로드할 폴더로 이동합니다.
2. 아래 명령어를 입력하여 프로젝트를 클론합니다:
   ```bash
   git clone [https://github.com/](https://github.com/)[본인깃허브아이디]/[레포지토리이름].git
클론이 완료되면 해당 폴더로 이동하여 VS Code를 엽니다:

Bash
cd [레포지토리이름]
code .
VS Code의 탐색기에서 index.html 파일을 우클릭하고 Open with Live Server를 클릭하면 브라우저에 플래너가 실행됩니다.

📖 사용 가이드 (User Guide)
화면 전환: 첫 화면에서 Day 또는 Night 아이콘을 클릭하여 모드를 선택합니다. (각 모드 하단의 버튼을 통해 교차 이동 및 홈 화면 이동이 가능합니다.)

일정 및 할 일 추가:

Schedule: 시간과 내용을 적고 + 버튼을 누르면 추가됩니다. 체크박스를 누르면 완료선이 그어집니다.

Top Tasks: 할 일을 적고 체크박스를 클릭하여 진행 상황을 관리하세요.

데이터 저장: 작성이 끝나면 반드시 우측 하단의 저장하기 버튼을 눌러주세요. (저장하지 않고 새로고침하면 날아갑니다!)

과거 기록 보기: 화면 좌측 하단의 캘린더에서 점(•)이 찍힌 날짜를 클릭하면, 그날 작성했던 기록을 한눈에 볼 수 있습니다.

🛠️ Technical Details (작동 원리)
☁️ 실시간 날씨 정보는 어디서 오나요?
본 플래너의 날씨 정보는 Open-Meteo API를 사용하여 실시간으로 가져옵니다.

Open-Meteo는 별도의 회원가입이나 API Key 발급 없이 무료로 사용할 수 있는 오픈소스 날씨 API입니다.

앱이 실행될 때 자바스크립트의 fetch() 함수를 통해 서울의 위도(37.5665)와 경도(126.9780)를 기준으로 현재 날씨, 최고/최저 기온, 강수 확률 데이터를 불러오며, 날씨 코드에 맞춰 화면의 아이콘(☀️, ☁️, 🌧️ 등)이 자동으로 변경되도록 구현되어 있습니다.

📅 날짜와 시간은 어떻게 매일 자동으로 세팅되나요?
플래너의 시간과 날짜는 서버가 아닌 사용자가 현재 접속 중인 기기(스마트폰, 컴퓨터)의 시스템 시간을 기준으로 세팅됩니다.

자바스크립트에 내장된 new Date() 객체를 사용합니다.

앱을 실행하는 순간 기기의 현재 연도, 월, 일을 가져와 화면의 달력, Today 날짜 텍스트, 그리고 이번 달의 진행률(Progress bar)을 실시간으로 계산하여 그려냅니다.

따라서 자정이 넘어가면 별도의 조작 없이도 자동으로 다음 날짜로 캘린더와 헤더가 업데이트됩니다.
